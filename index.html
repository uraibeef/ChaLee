Getting Started

ChaLee : An Education Robot for Everyone


ChaLee เป็นหุ่นยนต์เพื่อการศึกษาที่จะช่วยเสริมทักษะการเรียนรู้ของเด็กให้มีความสนุกมากขึ้น โดยพวกเรามีแนวคิดที่ว่าอย่างไรก็ตามการเรียนการสอนก็ต้องมีครู เพราะนอกจากจะให้ความรู้แล้วยังแนะนำสิ่งผิดถูกและคอยตอบข้อสงสัยของเด็กๆได้ เราจึงคิดหุ่นยนต์เพื่อเสริมการเรียนรู้ ไม่ใช่หุ่นยนต์ที่จะมาสอนเด็กๆแล้วทำหน้าที่แทนคุณครู ChaLee จะเป็นหุ่นยนต์อเนกประสงค์ สามารถใช้งานได้หลากหลาย ซึ่งคุณครูสามารถเลือกใช้วิธีต่างๆที่มากับหุ่นยนต์มาประยุกต์ให้เข้ากับการเรียนการสอนของคุณครูได้

ทำไมถึงต้องสร้าง ChaLee

ประเด็นปัญหา : การศึกษาเป็นรากฐานสำหรับทุกคน สิ่งต่างๆรอบตัวเราล้วนต้องใช้ความรู้และความเข้าใจในการเข้าถึง และเนื่องด้วยทุกวันนี้เทคโนโลยีได้เข้ามาทำให้โลกเปลี่ยนแปลงไป เราจำเป็นต้องเตรียมตัวทั้งการใช้ชีวิตและวิธีคิดเพื่อให้เข้ากับโลกอนาคต ดังนั้นการศึกษาจึงไม่อาจเป็นรูปแบบแบบเดิมๆ ได้อีกต่อไป เพราะรูปแบบการสอนแบบเดิมๆอาจไม่สามารถดึงความสนใจจากเด็กๆได้มากพอ นอกจากนี้เด็กๆตามชนบทหรือพื้นที่ห่างไกลอาจเข้าไม่ถึงการเข้ามามีบทบาทของเทคโนโลยี

ChaLee จะช่วยแก้ปัญหาอะไร

วิธีแก้ปัญหา : เราจึงพัฒนาสื่อการเรียนรู้ที่สอดแทรกเทคโนโลยีต่างๆเข้ามา เพื่อให้เด็กๆทุกคนสามารถเข้าใจและรู้ทันบทเรียนต่างๆควบคู่ไปกับเทคโนโลยีได้ ซึ่งการนำเทคโนโลยีเข้ามาสอนควบคู่ด้วย เด็กจะเริ่มเข้าถึงการเปลี่ยนแปลงของโลกปัจจุบัน ทำให้พวกเขารู้ว่าการศึกษาไม่เพียงแต่อยู่ตำราเรียนเท่านั้น พวกเขาสามารถเรียนรู้ได้ทุกที่เมื่อพวกเขาต้องการ เริ่มแรกเด็กๆอาจสงสัยถึงความเป็นไปของสิ่งที่เกิดขึ้น เป็นเหตุให้เกิดการตอบปัญหาด้วยตนเอง ซึ่งสิ่งนี้จะเป็นตัวกระตุ้นให้เกิดการอยากเรียนรู้ด้วยตัวเองต่อไป

แนวคิดการออกแบบ ChaLee

แนวคิดการออกแบบ : การเข้ามาของเทคโนโลยี จะทำให้ทักษะการเรียนรู้ที่เด็กๆควรรู้นั้นมีมากขึ้น โดยพวกเราคิดว่าการโคดดิ้งในเจนนี้เป็นเรื่องที่สำคัญ เริ่มแรกอาจเข้าใจเพียงแค่ตรรกะบางอย่าง จากนั้นจึงเริ่มเข้าสู่กระบวนการคิดอย่างมีระบบ ก่อนที่เริ่มเขียนโปรแกรม โดยสื่อการเรียนรู้นี้จะยังคงต้องมีครูกำกับดูแล เพื่อเป็นแนวทางต่างๆและตอบปัญหาของเด็กๆได้

Lesson Plans

การเรียนรู้ต่างๆ จะเรียนผ่านคุณครูเป็นหลัก โดยชาลีจะเป็นสื่อการเรียนการสอนที่คุณครูนำมาช่วยสอนเพื่อให้ความรู้และเสริมสร้างทักษะต่างๆให้กับเด็ก โดยหลักๆจะแบ่งออกเป็น 3 หมวด คือ หมวดคณิตศาสตร์ หมวดวิทยาศาสตร์ และหมวดศิลปะ

หมวดคณิตศาสตร์

จะมุ่งเน้นไปที่ตัวเลข รูปร่าง รูปทรง และตรรกะพื้นฐานต่างๆ เป็นการวางรากฐานให้เด็กๆคุ้นชินกับการใช้ทักษะไหวพริบในการแก้ไขปัญหา การเรียนจะยากขึ้นตามการนำมาประยุกต์ของคุณครู

หมวดวิทยาศาสตร์

จะมุ่งเน้นไปที่การใช้เซนเซอร์เพื่อให้เด็กสามารถมองเห็นภาพของการเรียนการสอนมากกว่าในหนังสือ โดยเซนเซอร์จะถูกนำมาอธิบายเพิ่มเติม ซึ่งขึ้นกับการประยุกต์ใช้กับบทเรียนนั้นๆ

หมวดศิลปะ

จะมุ่งเน้นให้อิสระกับเด็กๆ เริ่มแรกจะให้เด็กๆรู้จักพื้นฐานของศิลปะ เส้น สี จากนั้นจะสร้างเสริมจินตนาการเด็กๆด้วยเสียงเพลง อีกทั้งยังเพิ่มสนุกด้วยการแต่งเพลงเองอีกด้วย

Amazing Features.

สามารถประยุกต์ใช้งานได้หลากหลายรูปแบบ มาพร้อมกับ Sensor ต่างๆ ที่จะช่วยให้การเรียนรู้มีความสนุก ได้ประโยชน์มากยิ่งขึ้น

Tangible Programming

เรียนรู้กระบวนการคิดเชิงตรรกะ โดยไม่ต้องเขียนโปรแกรม (Without coding)

Blockly Programming

พัฒนาต่อยอดเป็นการเขียนโปรแกรมด้วยภาษา Blockly ได้ในอนาคต

Python Programming

ใช้ภาษา Python ในการควบคุม ใช้งานง่าย และเป็นที่นิยม

Brick Designed

Port เชื่อมต่อต่างๆถูกออกแบบมาให้สามารถเชื่อมต่อได้ง่าย คล้ายกับต่อตัวต่อเลโก้ ผ่านวิธีเชื่อมต่อด้วย Magnet Pogo Pin

Text-to-Speech

ใช้เทคโนโลยี Text-to-Speech ที่จะแปลงจากข้อความ(Text) เป็นเสียงพูดสังเคราะห์(Speech) เข้ามาช่วยในการแสดงผลด้วยเสียง

Mathematics

บูรณาการคณิตศาสตร์ ผ่านการแก้ปัญหา การคิดแบบมีระบบ การนำโจทย์ต่างๆมาประยุกต์ใช้ในการเล่นกับ ChaLee

Science

บูรณาการวิทยาศาสตร์ ด้วยการทดลงผ่านการประยุกต์ใช้งาน Sensor ต่างๆ

Art & Creativity

กระตุ้นให้เด็กเกิดความคิดสร้างสรรค์ จากการเรียนการสอนแบบใหม่นี้

Machine Vision

นำเทคโนโลยี Machine Vision ที่จะช่วยให้ ChaLee มองเห็นเหมือนมนุษย์ได้มาใช้งาน เพื่อให้เกิดความหลากหลายในการสอน

Sensor

สามารถเชื่อมต่อกับ Sensor ได้หลากหลายผ่านระบบการสือสารแบบ Serial Communication (UART, I2C, SPI)

3.IMU Sensor (Sense-Hat)
4.Soil Humidity Sensor
